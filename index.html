AR Sushi Menu

AR Sushi Menu is an application to display sushi with Augmented Reality. The main problem is that the traditional menu is not effective for consumers to make the right decision. With the Augmented Reality Technology can enhance the user experience when making the decision.

  • Unity3D, Vuforia, C# programming will be used in this project
  • The application is targeted at customer in Sushi King.
  • Application is using Marker-based tracking.
  • Application able to display a 3D Sushi.
  • Application able to let user interact with the menu through Augmented Reality.
